# SteamOS 3.8.0 概要・ダウンロード・インストール手順

このリポジトリは **SteamOS 3.8.0** の概要、ダウンロード方法、インストール手順をまとめた日本語ガイドです。  
Steam Deck をはじめ、ROG Ally や Lenovo Legion Go S などの携帯ゲーミング PC での利用を想定しています。

---

## 📌 1. SteamOS 3.8.0 の概要

**SteamOS 3.8.0** は Valve が提供する Arch Linux ベースのゲーミング向け OS です。

主な更新点:

- Linux カーネル **6.11 系**へ更新  
- Mesa グラフィックスドライバ更新  
- Steam Deck 以外の携帯ゲーミング PC でも動作報告  
- Windows より高いパフォーマンスを示すケースも存在  

---

## 📥 2. ダウンロード手順

### 2-1. インストールイメージの入手

以下の SteamOS 公開リポジトリにアクセスします。

https://steamdeck-images.steamos.cloud/steamdeck/20250320.1000/


### 2-2. ダウンロードするファイル

例: steamdeck-repair-20250320.1000-3.8.0.img.zip
 
ZIP を展開し、`*.img` を取り出します。

### 2-3. ライセンス確認

同ディレクトリの  
**「END USER LICENSE AGREEMENT FOR STEAM DECK® IMAGE」**  
を確認してください。

## 💾 3. インストールメディア作成（ブータブル USB）

### 必要なもの

- 8GB 以上の USB メモリ  
- Windows PC  
- Rufus（USB 書き込みツール）

### 手順

1. USB メモリを接続  
2. Rufus を起動  
3. 「ブートの種類」で `*.img` または `*.iso` を選択  
4. 設定は基本的に初期値で OK  
5. 「開始」をクリックして書き込み  

---

## 🖥 4. インストール手順（Steam Deck / 対応 PC）

### 4-1. USB から起動

1. USB を挿入  
2. 起動時に F2 / F12 / ESC / DEL などでブートメニューへ  
3. USB を選択して起動  

### 4-2. インストールメニュー

- **Reimage**  
  → 初期化して再インストール  
- **Reinstall**  
  → 設定やデータを残したまま修復  

### 4-3. 初期設定

1. 再起動後、セットアップ画面が表示  
2. 言語・キーボード・ネットワークを設定  
3. Steam アカウントでログイン  

---

## 🔄 5. SteamOS 3.8 への更新

### 5-1. 開発者モードを有効化

1. 設定 → システム / 開発者  
2. 開発者モードを ON  

### 5-2. アップデートチャンネル変更

1. アップデート設定を開く  
2. チャンネルを **Main（メイン）** に変更  
3. アップデートを適用  

---

## ⚙️ 6. 性能と利用上のポイント

- Windows より高いゲーム性能を示す例あり  
- デスクトップモードで Linux として利用可能（制限あり）  
- 他社製ゲーミング PC での導入事例が増加  

---

## ⚠️ 7. 注意事項

- SteamOS は Steam Deck 向けに設計されています  
- 他社製デバイスは **動作保証外**  
- インストール時にデータは消失します  
- 公開 URL やバージョンは変更される可能性があります
